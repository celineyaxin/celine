# 大纲
## filter_simple.py
提取单个年报文件中的指定内容

## filter.py
提取所有年报文件中的制定内容

## merge.py
提取出的文件内容进行合并

## frequence.py
* 查看根据文本内容的分词结果
* 对一些高频词进行stop_words以及user_dic的补充
* 确定min_freq的数值
* 接下来根据调整的结果可以直接构造jason文件了





## jasonsave.py
根据分词的结果保存进jason文件

## word2vec_train
* 根据分词结果对文本进行训练
* 能够更好的查看epoch的迭代信息

## word2vec_test
看训练的结果

## output
结果输出
